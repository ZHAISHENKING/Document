### Code码

> 0、1、2、4分别代表成功、错误请求、数据库操作错误、无效验证(jwt无效)

```
{
        "code": 0,
        "data": data,
        "msg": "请求成功"
}
{
        "code": 1,
        "data": '',
        "msg": msg
}
{
        "code": 2,
        "msg": "数据库操作错误"
}
{
        "code": 4,
        "data": data,
        "msg": "无效验证"
}
```
